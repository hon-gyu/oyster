# Obsidian Title Syntax Specification

Obsidian titles are the file name of the markdown file.

Obsidian does not allow these characters in note titles:
- `#` (heading separator)
- `^` (block reference)
- `[` `]` (link delimiters)
- `|` (display separator)
- `\` `/` (path separators, though `/` can appear in paths)
